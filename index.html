The learning is intense guys! So much so that I've built a whole new version of the site. This won't be a regular occurence - I'll be making regular tweaks directly onto the site, but when I'm making more major changes, I'll version.

You'll be able to find old versions of the site here (or by clicking on the 'Past lives' link in the header navigation bar.

Here are the things that're new in Version 2:

  • The typewriter effect for the header (using this).
  • The colour gradient effect for the background (using the same colour-scheme as this gif that featured in Version 1's header, selected using this eyedropper Chrome extension).
  • Smooth scrolling - you know how when you're on a Wikipedia page, and you select a link to another part of the page, and it jumps straight there? I wanted to smooth this movement out to make it slicker. Again, I've copycatted my code from the wonderful CSS-Tricks.
  • The navigation bar items just beneath the header fade in! An incredible trick Stack Overflow taught me.
