Hacking 101

You wouldn't think that developers at Twitter would be interested in teaching people how to hack. Turns out, they are.

Within half an hour of my first Code First: Girls session at Twitter HQ, we were taught how to edit the text of a webpage*:

  1. Highlight the text you want to edit, right click and select Inspect.

  2. The code that makes up the webpage will now magically appear either adjacent to, or above/below the visible webpage. You'll be taken directly to the section of the code that corresponds to the text you've selected.

  3. Double click the text you'd like to edit, which will open a text editor, and type in your alternative text.

  4. Press Enter and watch your webpage *magically transform*.

  5. Punk as appropriate.

*If only this were true in the way you're imagining. Sadly, editing the text this way only alters the local version of the page on your individual computer, rather than the remote version that exists on the server.
